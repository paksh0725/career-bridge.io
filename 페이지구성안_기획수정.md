# Career Bridge 페이지 목업 작업 지시서

## 작업 원칙

- 이번 작업은 실제 기능 개발이 아니라 페이지 목업 고도화 작업이다.
- 기존 페이지에 이미 기능형으로 구현되어 있거나, 화면상 표현되어 있는 항목은 유지한다.
- 기존 기능을 삭제하거나 구조를 크게 바꾸지 않는다.
- 아래 항목 중 현재 페이지에 없는 기능만 화면 요소, 더미 데이터, 카드, 탭, 모달 형태로 추가한다.
- 서버 연동, 실제 로그인, 실제 DB 저장, 실제 암호화, 실제 OCR 처리는 구현하지 않는다.
- 필요한 경우 “처리 중”, “승인 완료”, “검수 필요”, “암호화 적용”, “OCR 완료” 같은 상태 배지로 기능 의도를 표현한다.
- 수치는 실제 데이터가 아니라 목업용 더미 수치로 넣어도 된다.

## 공통 목업 지시

- 모든 역할 화면에서 기존 톤앤매너, 색상, 카드 스타일, 사이드바/탭 구조를 최대한 유지한다.
- 이미 존재하는 검색, 메시지, 번역, 커뮤니티 기능은 그대로 둔다.
- 새 기능은 별도 탭, 카드, 요약 패널, 모달, 드로어 중 기존 UI 패턴에 맞는 방식으로 추가한다.
- “AI OCR”, “암호화”, “접근 로그”, “권한 관리”는 실제 기능이 아니라 관리자/담당자가 이해할 수 있는 화면 표시 수준으로 구현한다.
- 기능 이름만 나열하지 말고, 담당자가 볼 수 있는 구체적인 지표와 상태값을 함께 보여준다.

## 작업 우선순위

```text
1순위
├─ 기관 담당자 화면 고도화
├─ 기업 담당자 화면 고도화
└─ 관리자 화면 고도화

2순위
├─ 유학생 화면 보강
├─ AI OCR 상태 표시
└─ 암호화/보안 상태 표시

3순위
├─ 리포트/지표 카드 추가
├─ 권한/접근 로그 목업 추가
└─ 메시지 템플릿/상담 이력 등 보조 화면 추가
```

## 유학생 화면 작업 지시

### 유지할 기능

- 프로필, 이력 관리, 문서 관리, 공개 설정, 기업 검색, 커뮤니티, 메시지, 번역 서비스는 기존 화면이 있으면 그대로 유지한다.
- 문서 업로드, 문서 상세, 기업 상세, 메시지 보내기 같은 기존 모달이 있으면 삭제하지 않는다.

### 없는 경우 추가할 목업

- [ ] 프로필 완성률 카드 추가
  - 예시 수치: `87%`
  - 하위 항목: 기본 정보, 이력, 문서, 공개 설정, 번역 프로필

- [ ] 취업 가능 조건 요약 카드 추가
  - 체류 자격
  - 졸업 예정일
  - 취업 가능 시점
  - 비자 전환 가능성
  - 희망 직무 적합도

- [ ] 문서 인증/OCR 상태 카드 추가
  - 이력서: OCR 완료
  - 성적증명서: 검수 필요
  - 재학증명서: 인증 완료
  - 추천서: 미등록
  - OCR 인식률 예시: `94.2%`

- [ ] 기업 공개용 프로필과 기관 관리용 프로필 구분 표시
  - 기업 공개 정보: 이름, 학교, 전공, 언어, 희망 직무, 공개 문서
  - 기관 관리 정보: 체류 자격, 상담 이력, 문서 제출 상태, 비자 만료일

- [ ] 개인정보 보호 안내 카드 추가
  - 문서와 민감 정보는 암호화되어 저장된다는 안내
  - 기업은 공개 허용 정보만 열람 가능하다는 안내

### 유학생 화면에 보여줄 수치

- 프로필 완성률
- 문서 인증 완료율
- 기업 열람 수
- 메시지 응답률
- 추천 기업 수
- 지원 가능 공고 수
- 번역 사용 횟수
- OCR 인식 성공률

## 기관 담당자 화면 작업 지시

### 유지할 기능

- 학생 조회, 지원 현황 분석, 커뮤니티, 안내 메시지 기능은 기존 화면이 있으면 그대로 유지한다.
- 학생 상세 모달, 안내 메시지 발송, 지원 대상 등록 기능이 있으면 삭제하지 않는다.

### 기관 화면의 방향성

- 기관은 채용 성과보다 학생 보호, 행정 지원, 상담 관리, 문서 검수, 비자/체류 관리가 중요하다.
- 기업 화면과 다르게 “관리 필요 학생”, “지원 누락”, “상담 필요”, “문서 미제출”, “비자 만료 임박”을 강조한다.

### 없는 경우 추가할 목업

- [ ] 관리 필요 학생 큐 추가
  - 비자 만료 임박
  - 문서 누락
  - 상담 미완료
  - 취업 희망 높음
  - 한국어 역량 부족
  - 예시 상태: `긴급`, `확인 필요`, `상담 예정`

- [ ] 비자 만료 예정자 알림 카드 추가
  - D-30 대상자 수
  - D-60 대상자 수
  - 체류 자격별 학생 수
  - 만료 임박 학생 리스트

- [ ] 문서 미제출자 리스트 추가
  - 성적증명서 미제출
  - 재학증명서 미제출
  - 이력서 미등록
  - 추천서 미등록

- [ ] 상담 이력 관리 목업 추가
  - 최근 상담일
  - 상담 유형: 취업, 비자, 생활, 학업
  - 다음 상담 예정일
  - 상담 상태: 예정, 완료, 재상담 필요

- [ ] 취업 지원 단계 관리 추가
  - 대상자 선정
  - 프로필 보완
  - 문서 검수
  - 기업 추천
  - 면접 준비
  - 사후 관리

- [ ] 기관 문서/OCR 검수 화면 추가
  - 원본 문서명
  - OCR 추출 정보
  - OCR 신뢰도
  - 수동 확인 필요 여부
  - 승인/반려 버튼 목업

- [ ] 기관 성과 리포트 카드 추가
  - 취업지원 대상자 수
  - 상담 전환율
  - 상담 완료율
  - 문서 제출률
  - 인증 완료율
  - TOPIK 4급 이상 비율
  - 정주 희망률
  - 지원 프로그램 참여율

## 암호화 및 보안 목업 작업 지시

### 실제 구현 범위 아님

- 실제 암호화 로직은 구현하지 않는다.
- 목업에서는 암호화 적용 상태, 접근 권한, 감사 로그, 보안 이벤트를 화면에 표시한다.

## AI OCR 및 AI 기능 목업 작업 지시

### 실제 구현 범위 아님

- 실제 OCR 엔진, AI 모델, 번역 API는 구현하지 않는다.
- 목업에서는 OCR 처리 상태, 추출 결과, 신뢰도, 검수 상태를 더미 데이터로 보여준다.

### 없는 경우 추가할 화면 요소

- [ ] 문서 OCR 상태 표시
  - 이력서 OCR
  - 자기소개서 OCR
  - 성적증명서 OCR
  - 재학/졸업증명서 OCR
  - 자격증 OCR
  - 추천서 OCR

- [ ] OCR 자동 추출 결과 카드
  - 이름
  - 학교/기관
  - 전공
  - 학점
  - 발급일
  - 자격명
  - 경력/프로젝트 키워드
  - 언어 능력

- [ ] OCR 검증/보정 화면
  - OCR 신뢰도 점수
  - 낮은 신뢰도 항목 표시
  - 수동 확인 필요 표시
  - 문서 위변조 의심 표시
  - 관리자 검수 큐

- [ ] AI 매칭/추천 표시
  - 후보자-직무 매칭 점수
  - 기업 추천
  - 지원 필요 학생 추천
  - 채용 가능성 예측

- [ ] AI 번역/요약 표시
  - 이력서 다국어 번역
  - 메시지 번역
  - 문서 요약
  - 담당자용 핵심 정보 요약

## 최종 확인 기준

- [ ] 기존 기능형 화면은 삭제되지 않았다.
- [ ] 없는 기능만 추가 목업으로 들어갔다.
- [ ] 기관 화면은 학생 관리/상담/비자/문서 검수 중심으로 차별화되었다.
- [ ] 기업 화면은 후보자 매칭/채용 파이프라인/성과 지표 중심으로 차별화되었다.
- [ ] 관리자 화면은 운영자/경영자/마케터 관점의 지표가 모두 들어갔다.
- [ ] 암호화와 보안 항목이 화면에서 확인 가능하다.
- [ ] AI OCR 항목이 화면에서 확인 가능하다.
- [ ] 실제 기능 연동 없이 목업 데이터만으로 화면 의도가 전달된다.
