## 페이지별 내용

### `index.html` - 메인 랜딩

- 서비스 소개 히어로: 유학생 커리어 통합 관리 메시지, 유학생 시작하기, 기업 담당자 포털 버튼
- Product Demo: 실제 서비스 화면 미리보기
- 주요 기능 소개: 통합 프로필 관리, 기업·학교 매칭, 다국어 번역
- 핵심 기능 카드: HR 인재 검색, 학생 프로필/이력/문서 관리, 메시지 연결, 번역 지원
- 사용자 유형 진입 카드: 유학생, 기관 담당자, 기업 인사담당자
- 운영 지표: 등록 유학생, 제휴 기업, 지원 언어, 프로필 완성률
- CTA: 무료 회원가입, 기업 담당자 포털 바로가기
- 모달: 로그인, 회원가입, 서비스 화면 미리보기

### `login.html` - 단독 로그인

- 이메일/비밀번호 입력
- 로그인 버튼
- 홈으로 이동
- 회원가입 링크
- 기관 담당자/기업 담당자 포털 바로가기

### `student.html` - 유학생 대시보드

- 프로필: 기본 정보, 관심 분야/계획, 언어 능력, 자기소개
- 이력 관리: 학력/경력/자격 이력, 관련 문서 업로드/보기/다운로드
- 문서 관리: 등록 문서 목록, 문서 업로드, 문서 상세 보기
- 공개 설정: 프로필 공개 범위, 검색 노출 설정
- 기업 검색: 기업 리스트, 조건 필터, 기업 상세, 기업 담당자에게 메시지 발송
- 커뮤니티: 카테고리 필터, 인기글 TOP 3, 글쓰기, 투표
- 모달: 프로필 편집, 이력 추가, 문서 등록, 문서 상세, 기업 상세, 메시지 보내기, 글 작성, 투표

### `messages.html` - 유학생 메시지

- 메시지함: 전체/받은 메시지/보낸 메시지 탭
- 메시지 검색
- 대화 목록
- 채팅 상세: 상대 정보, 메시지 내용, 메시지 입력/발송
- 번역 기능: 채팅 상단 번역 버튼, AI 번역으로 발송 안내, 번역 결과 발송 모달
- 새 메시지 작성 모달

### `translation.html` - 번역 서비스

- 한국어 기준 영어/중국어 번역
- 번역 방향 전환
- 원문 입력, 번역 결과 출력, 글자 수 카운트
- 빠른 템플릿: 이력서 문구, 자기소개서 문구, 메시지 문구
- 프로필 3개 언어 동시 번역
- 최근 번역 내역

### `institution.html` - 기관 담당자 포털

- 학생 조회: 조건 검색/필터, 학생 카드 목록, 선택 관리
- 지원 현황 분석: 국적별 유학생 분포, 한국 정주 의사 현황, 인기 전공 TOP 5, TOPIK 등급 분포
- 커뮤니티: 취업정보 공유, Q&A, 투표, 인기글 TOP 3, 글쓰기
- 모달: 학생 상세, 기관 안내 보내기, 지원 대상 등록, 글 작성, 투표

### `inst-messages.html` - 기관 담당자 메시지

- 지원 대상 메시지함
- 유학생 검색
- 대화 목록
- 채팅 상세: 학생 정보, 메시지 내용, 메시지 입력/발송
- 메시지별 번역 토글
- 기관 포털의 학생 조회/지원 현황/커뮤니티로 복귀 가능

### `company.html` - 기업 담당자 포털

- 인재 검색: 채용 조건 기반 후보자 검색/필터, 유학생 카드 목록, 선택 관리
- 인사이트 분석: 국적별 유학생 분포, 한국 정주 의사 현황, 인기 전공 TOP 5, TOPIK 등급 분포
- 커뮤니티: 취업정보 공유, Q&A, 투표, 인기글 TOP 3, 글쓰기
- 모달: 유학생 상세, 메시지 보내기, 일괄 메시지 발송, 글 작성, 투표

### `company-messages.html` - 기업 담당자 메시지

- 지원자 메시지함
- 지원자 검색
- 대화 목록
- 채팅 상세: 유학생 정보, 메시지 내용, 메시지 입력/발송
- 메시지별 번역 토글
- 기업 포털의 인재 검색/인사이트/커뮤니티로 복귀 가능

### `admin.html` - 관리자 콘솔

- 운영 대시보드: 플랫폼 현황, 처리 대기 업무, 보안 이벤트, 주간 가입 추이, 핵심 지표
- 보안 · 인증 관리: DID/SBT 인증 승인 대기, 발급기관 화이트리스트, 보안 이벤트 로그
- 회원 관리: 가입/탈퇴 현황, 회원 유형별 분포, 방문자 추이, 그룹별 평균 방문 횟수
- 콘텐츠 관리: 신고글 처리, 게시글 승인
- 파트너 · 협력기관 관리: 중국 유학원 네트워크, KT 통신 연동, 글로벌 협력 현황
- 통계 · 리포트: 가입자 추이, 국가별 분포, 인증 완료율, AI 매칭 성사율, PDF/CSV 내보내기
- 배너 · 홍보 관리: 배너 목록, 노출 위치/기간/상태, 추가/편집/삭제
- 시스템 · 권한 관리: 관리자 역할, 권한 편집, 감사 로그
- 기업 · 기관 권한 관리: 대학/기관/기업 권한 등급, 인증 발급 가능 여부, 유학생 대표 권한
- 모달/드로어: 인증 상세, 발급기관 추가, 사업자 확인, 배너 편집, 역할 편집, 상세 드로어

## 역할별 핵심 구조 요약

```text
유학생
└─ 프로필/이력/문서 관리
   ├─ 공개 설정
   ├─ 기업 검색
   ├─ 커뮤니티
   ├─ 메시지
   └─ 번역 서비스

기관 담당자
└─ 학생 조회/지원 관리
   ├─ 지원 현황 분석
   ├─ 커뮤니티
   └─ 안내 메시지

기업 담당자
└─ 인재 검색/후보자 관리
   ├─ 인사이트 분석
   ├─ 커뮤니티
   └─ 지원자 메시지

관리자
└─ 플랫폼 운영 관리
   ├─ 보안/인증
   ├─ 회원/콘텐츠
   ├─ 파트너/통계
   ├─ 배너/홍보
   └─ 시스템/권한
```
